### S5 实现功能
- **任务创建**: 根据选中的时间线列表 + ExportSettings 创建 ExportJob 列表
- **命名集成**: 每条时间线通过命名引擎生成唯一文件名
- **真实导出流程**:
  1. 切换到目标时间线 (SetCurrentTimeline)
  2. 配置渲染参数 (SetRenderSettings)
  3. 添加到渲染队列 (AddRenderJob)
  4. 批量启动渲染 (StartRendering)
- **Mock模式**: 无达芬奇环境时模拟导出流程
- **进度回调**: current/total/timeline_name 实时反馈
- **日志回调**: 每步操作文字记录
- **错误处理**: 单条失败不影响其他任务
- **状态追踪**: pending → queued → rendering → done/failed
- **取消支持**: 随时终止导出、清除渲染队列
- **导出完成弹窗**: 成功/失败数量统计

### ExportEngine API
```
engine = ExportEngine()
engine.create_jobs(selected_timelines, settings)  # 创建任务
engine.execute(settings, progress_cb, log_cb)     # 执行导出
engine.cancel()                                    # 取消
engine.get_status()                                # 获取状态
```
