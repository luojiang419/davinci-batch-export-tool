### S4 实现功能
- **9种命名变量**: {timeline} {date} {time} {index} {project} {resolution} {fps} {format} {codec}
- **变量按钮插入**: 点击变量按钮在光标位置插入变量
- **实时预览**: 编辑模板时自动更新预览，格式切换时联动更新扩展名
- **文件名清理**: 自动移除 `<>:"/\|?*` 等非法字符
- **序号格式化**: 两位数补零 (01, 02, ...)
- **单例引擎**: get_naming_engine() 全局复用

### 命名模板示例
```
{timeline}_{date}        → 开篇动画_20260601.mp4
{project}/{timeline}     → 我的项目/开篇动画.mp4
{timeline}_{fps}fps_{date} → 开篇动画_24fps_20260601.mp4
```
