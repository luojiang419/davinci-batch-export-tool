### S3 实现功能
- **ExportSettings 数据模型**: 完整结构化保存所有导出参数，含 Resolve API 映射方法
- **高级视频参数**: 数据级别(Auto/Video/Full)、色彩空间(Rec.709/2020等)、编码档次
- **自定义分辨率**: 选择"自定义"时显示宽×高输入框，带数字验证
- **关键帧间隔**: SpinBox 0-300帧，0=自动
- **Alpha通道**: 复选框支持导出透明通道
- **跳过重新编码**: 可能时直接复制流
- **音频比特率**: 128/192/256/320 kbps 可选
- **collect_settings()**: 从UI收集所有设置到 ExportSettings 对象
- **格式联动**: 格式切换时自动更新编码器选项
- **数据级别/色彩空间/编码档次** 完整下拉选项

```
格式: MP4/QuickTime/AVI/MXF OP-Atom/MXF OP1A
```
